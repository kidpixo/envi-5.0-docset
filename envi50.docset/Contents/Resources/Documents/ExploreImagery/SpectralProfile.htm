Spectral Profile

Use the Spectral Profile to plot the spectrum of all bands for the selected pixel. You can extract spectra from any multispectral dataset. ENVI uses header information to scale the plot. If multiple layers are displayed in the Image window view, ENVI plots a spectral profile from each layer. To turn the plot of for an individual layer, disable the check box for that layer in the Layer Manager.

Select Display > Spectral Profile from the menu bar to open the Spectral Profile dialog.

Extract Spectra

  • To extract spectra, click the Select button on the toolbar, then click on a pixel in the Image window to plot it in the Spectral Profile. When you click on a different pixel, the plot updates with the spectrum for that pixel.
  • Press and hold the Shift key as you click on pixels to overplot spectra.
  • Click and drag the cursor to browse the spectra under the cursor. Press and hold the Shift key as you drag the cursor to overplot spectra.
  • The X axis (Wavelength) is determined by the first spectrum that is plotted. The Y axis (Value) is auto-scaled by default unless you right-click on the plot and select Auto Scale Y-Axis to disable auto-scaling, or if you have a Z plot range defined.

Note: X/Y pixel coordinates in ENVI -displayed spectra are one less than those displayed in an ENVI Classic Z profile. For example, a pixel location of [X:165, Y:73] in ENVI yields the same spectrum as location [X:166, Y:74] in ENVI Classic .

Plot Controls

  • Click on the arrow on the right side of the Spectral Plot dialog to expand the dialog panel. The panel shows a list of spectra included in the plot.
  • To change the display properties of the plot or a spectrum, double-click in the plot, or on a specific spectrum in the plot, or on the spectrum listed in the expanded panel. The Properties dialog appears for you to edit the settings.
  • Zoom in and out of the plot by clicking in the plot, then rolling the mouse wheel. Or, press the Ctrl key and draw a box around the area to zoom into. Right-click in the plot and select Reset Plot Range to return to the plot to full view.
  • Pan the plot by pressing the Shift key and dragging inside the plot.
  • To display an X-Y line cursor, click in the plot. Then click on spectrum in the plot, or expand the dialog panel and select the spectrum from the list. As you move the cursor over the plot, the line cursor snaps to the closest data point . The x and y values of the point appear in the lower-right corner of the dialog. To close the X-Y line cursor, click outside the plot.
  • To remove all spectra from the plot, right-click and select Remove All . To remove a spectrum or selected spectra from the plot, right-click on a spectrum or use Ctrl +click to more than one, then right-click and select Remove Selected Spectrum .
  • To save the plotted spectra to a spectral library, click the Save to Spectral Library button provided on the expanded dialog panel.

Drag and Drop Spectra to the Spectral Library Viewer

You can drag and drop spectra between the Spectral Library Viewer and the Spectral Profile for analysis:

  1. Open a spectral library file in the Spectral Library Viewer.
  2. Click the arrow on the right side of the Spectral Profile dialog and the Spectral Library Viewer dialog to expand the panels.
  3. From the Spectral Profile expanded panel, select a spectrum, or use Ctrl +click to multi select.
  4. Drag the selected items to the expanded panel of the Spectral Library Viewer dialog. The spectra appear in the corresponding plot.

Additional Controls

The button controls at the bottom of the dialog do the following:

  • Click Print to print the plot.
  • Click Save As to open the Save As dialog and save the plot.
  • Click Properties to open the Properties dialog and edit the attributes of the plot lines. Double-clicking in the plot also opens the Properties dialog.
  • Click Reset Axis Range to change the range for the x/y axis.
  • Click Edit to select from a drop-down list of options.
  • Click Annotations to select from a list of options to annotate the plot.
