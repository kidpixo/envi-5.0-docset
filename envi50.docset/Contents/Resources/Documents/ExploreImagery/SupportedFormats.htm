Supported File Types

Use the File > Open menu option to open data files from the following formats and sensors.

  • CARTOSAT-1 ( *.tif )
  • DPPDB ( *.ntf )
  • DTED ( *.dt0 ; *.dt1 ; *.dt2 )
  • ENVI Annotation ( *.ann ; *.anz )
  • ENVI Raster ( *.hdr )
  • ENVI Vector ( *.evf )
  • ERDAS IMAGINE ( *.ige ; *.img )
  • Esri® Grid ( hdr.adf )
  • Esri Layer ( *.lyr )
  • Esri Shapefile ( *.shp )
  • GeoEye-1 ( *.ntf ; *.tif ; *.txt )
  • IKONOS ( *.ntf ; *.tif ; *.txt )
  • JPEG ( *.jpg ; *.jpeg )
  • JPEG2000 ( *.jp2 ; *.j2k )
  • KOMPSAT-2 ( *.tif ; *.xml )
  • Landsat GeoTIFF with Metadata ( *.met ; *.tif ; *.txt )
  • MrSID ( *.sid )
  • NITF & NSIF ( *.ntf ; *.nitf ; *.nsf )
  • OrbView-3 ( *.ntf ; *.tif )
  • QuickBird ( *.ntf ; *.tif ; *.til )
  • RapidEye ( *.ntf ; *.tif ; *.xml )
  • Region of Interest
  • Spectral Library ( *.asd ; *.msl ; *.rad ; *.sli )
  • SPOT DIMAP ( *.dim )
  • SRTM DEM ( *.hgt )
  • TFRD ( *.tfd ; *.isd )
  • TIFF ( *.tif ; *.tiff )
  • WorldView-1 and -2 ( *.ntf ; *.tif ; *.til )

For all other data formats and sensors, see the following sections:

  • Generic Image Formats
  • Image Processing Software Formats
  • Military Formats
  • Other Formats
  • Sensors

Note: ENVI does not read imagery with 64-bit long integer (signed or unsigned) data.

Generic Image Formats

Open the following data files from the File > Open As > Generic Formats menu.

  • ASCII
  • BMP
  • JPEG
  • JPEG2000
  • HDF
  • MrSID
  • PDS
  • PICT
  • PNG
  • SRF
  • TIFF/GeoTIFF
  • XWD

Image Processing Software Formats

Open the following data files from the File > Open As > IP Software menu:

  • ArcView® Raster
  • ECW
  • ERDAS IMAGINE
  • ER Mapper
  • Esri® GRID
  • PCI

Military Formats

Open the following data files from the File > Open As > Military menu:

  • CADRG
  • CIB
  • DPPDB (NITF license required)
  • NITF and NSIF (NITF license required)
  • TFRD (TFRD license required)
