Connection Authentication

If a username and password are required to log in to a server or geodatabase, the Connection Authentication dialog appears. Perform the following steps:

  1. Enter a Username .
  2. Enter a Password . Passwords are displayed as a series of asterisks representing each character you type.
  3. Click OK in the Connection Authentication dialog to attempt a connection using the specified login information. If the connection fails, an error message appears and the Connection Authentication dialog remains open so that you can reenter the login information.

Note: Your username and password are retained for as long as the server or dataset is open.
