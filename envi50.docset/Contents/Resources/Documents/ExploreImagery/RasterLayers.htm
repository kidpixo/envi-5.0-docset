Raster Layers

Raster layers generally refer to images , including multispectral, panchromatic, digital elevation models (DEMS), and masks. For information about classification images, see Class Layers .

Manage Raster Layers

The Layer Manager lists all displayed files in a tree view. Raster layers that are open but are not displayed in an Image window view are listed only in the Data Manager. If you created multiple views in the Image window, the layers for each view are grouped under View entries. Each time you open a new data file, the layer appears at the top of the Layer Manager and becomes the selected layer . The purpose of a selected layer is to allow you to apply a display enhancement (brightness, contrast, transparency, etc.) to that layer only, without affecting other layers.

  • To select a layer, click its name in the Layer Manager.
  • To hide a layer in a view, disable the check box for that layer in the Layer Manager. To show the layer, enable the check box.
  • To remove a layer from the Layer Manager, right-click the layer name and select Remove . This removes the layer from the Layer Manager, but the file remains open and you can access it for redisplay from the Data Manager.
  • To change the RGB combination displayed for a multispectral image, right-click on a layer name in the Layer Manager and select Change RGB Bands . To assign an RGB combination, click the band name to assign to red in the Change Bands dialog, then repeat to make the green and blue selections. Click OK .

  • To change the color table for a grayscale image, right-click on the layer name in the Layer Manager and select Change Color Table . Choose Greyscale , Blue/White , Red Temperature , Blue/Green/Red/Yellow , or Rainbow as pre-defined color tables, or select More to use the ENVI Color Table dialog to create your own color table.

    Note: This option is not available for single-band layers that contain a pre-defined color palette, such as ENVI classification images.

  • To zoom to a 1:1 (100%) scale for a given layer, right-click on the layer name in the Layer Manager and select Zoom to Layer > Full Resolution .
  • To zoom to the full extent of a given layer, right-click on the layer name in the Layer Manager and select Zoom to Layer . This centers and displays the entire layer.
  • Selecting Full Extent from the Zoom drop-down menu or clicking the Zoom to Full Extent button in the main toolbar zooms to the combined extents of all layers for the current view.
  • If a raster layer is georeferenced, the Zoom To drop-down list in the main toolbar displays options for Map Scale (left image) or Pixel Scale (right image):

    For pixel-based raster layers (with no georeferencing), the Zoom To drop-down list only displays options for Pixel Scale.

Map Information

  • The first raster layer loaded into the Layer Manager that contains map information is the base layer . This layer defines the map projection and resolution.
  • Subsequent layers loaded into the Layer Manager are reprojected on-the-fly to match the base layer projection.
  • If another raster dataset loaded into the layer has no associated map information, ENVI assumes it matches the projection and pixel size of the base layer, so it does not resample or reproject the data.
  • You should always load images with map information or with higher resolution first. If you happen to load them the other way around, ENVI reprojects the higher resolution image to match the projection of the lower resolution image, without losing resolution.
  • Images that only contain rational polynomial coefficients (RPCs) cannot be used as a base layer when using other layers with map projections because the RPCs are not rectified standard projections.
  • If you load two layers with RPC information, the base projection will be UTM.
  • If an image contains both RPC and standard map projection information, ENVI will not save the RPC information to the file header when you save the image to ENVI raster format.

Change a Color Table

You can apply a color table to a grayscale image when you right-click the layer in the Layer Manager and select Change Color Table . You can choose from several pre-define color tables, or click More to open the Change Color Table dialog. In the dialog you can pick from more pre-defined IDL color tables, and you can work in RGB (red-green-blue), HLS (hue-lightness-saturation), HSV (hue-saturation-value), or CMY (cyan, magenta, yellow) color spaces to define your own color table.

  • When you open the dialog, the Selected Color Table color bar shows the current color table for the image. When you modify the color table, the color bar updates to show the changes and the colors are applied to the image for preview. To accept the new color table and close the dialog, click OK .
  • Click Reverse to reverse the order of the color gradient.
  • To define your own color table, select RGB , HLS , HSV , or CMY from the drop-down.
  • Enable the check box next to the slider bar of the color space element you want to change. The two remaining sliders represent horizontal and vertical axes of the currently displayed color space and remain constant; only the slider with the enabled check box can change the current color space.
  • Move the slider bars for an element to adjust the values of the color space.
  • In the color space window, click and drag the end points of the interpolation line to change the current color table. The color table will contain the colors that fall beneath this line.
