Count Features

Feature Counting Tutorial

Features are user-defined objects that can be modeled or represented using geographic data sets. Examples include roads, buildings, and water bodies.

Use the Feature Counting tool to mark features in raster and vector data. You can mark multiple feature types and add descriptions of them. You can also mark individual features, and you can create feature type categories to help differentiate the markings between feature types.

Mark, Save, and Restore Features

  • Click the Feature Counting button. In the Feature Counting dialog, click the Edit Feature Type Properties button. In the Properties dialog, enter a Name and Description for the feature type you want to mark. The description applies to all features you mark as this feature type. If desired, you can change other properties set for the feature type in the dialog.
  • In the Image window view, click on the features to mark. A row is added to the Feature Counting dialog, containing the latitude/longitude or pixel coordinates of the feature, and a Feature Counting layer is added to the Layer Manager. The layer will contain all feature types.
  • To create a new feature type, click the Add Feature Type button in the Feature Counting dialog then right-click on the new feature type and select Edit Properties to add a name and description.
  • To save the feature counting layer ( .dbz ), click the Save Features button.
  • To restore a saved feature counting layer, click the Restore Features button in the Feature Counting dialog.

Jump to Features

In the Feature Counting dialog:

  • Click on the Jump to First Feature button to go to the first feature listed in the data table.
  • Click on the Jump to the Previous Feature button to go to the feature listed above the selected feature in the data table.
  • Click on the Jump to the Next Feature button to go to the feature listed below the selected feature in the data table.
  • Click on the Jump to Last Feature button to go the last feature listed in the data table.
  • With the Enable Table Goto check box enabled, click on a row in the data table to center the Image window view over the feature. Disable the check box if you do not want to jump to features by clicking on a table row. Clicking the Jump to buttons will still result in the Image window view centering over features.

View the Feature Report

  • Select the Report tab in the Feature Counting dialog, then click Update Report . The Report tab is updated with information about all features types that were marked.
  • To clear the report data, click Clear .
  • To save the feature counting report as a text file, click Save .

Zoom Options

  • To zoom to the full extent of a given feature layer, right-click on the layer name in the Layer Manager and select Zoom to Layer . This centers and displays the entire layer.
  • Selecting Full Extent from the Zoom drop-down menu or clicking the Zoom to Full Extent button in the main toolbar zooms to the combined extents of all layers for the current view.
