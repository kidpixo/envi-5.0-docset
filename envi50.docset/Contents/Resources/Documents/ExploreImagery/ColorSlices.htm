Raster Color Slices

Use the Raster Color Slices tool to select data ranges and colors to highlight areas of an image. You can save the color slices visualization to a classification or to a shapefile.

  1. Generate raster color slices using one of the following:
    • In the Layer Manager, right-click the raster name and select Raster Color Slices . Select the input file and band in the Select Input File dialog.
    • In the Data Manager, right-click on the band to use as input and select Load Raster Color Slices .
    • From the Toolbox, select Classification > Raster Color Slices . Select the input file and band in the Select Input File dialog.
  2. The Edit Raster Color Slices dialog appears and includes a histogram of the raster color slices using the following default settings:
    • Number of slices = 16
    • Color table = Rainbow
    • Minimum and maximum data ranges, calculated from the image
  3. Keep the default settings, or edit them as needed (described in Edit Color Slices below). Click OK to create the raster color slices.

ENVI adds the raster color slices into the Layer Manager as filename - band , where filename - band is the name of the file and band used to create the raster color slices. The raster color slice layer loads in the Image window view.

  • To hide a raster color slice in the view, disable the check box for that color slice in the Layer Manager. To show the color slice again, enable the check box.
  • To hide all raster color slices, right-click the Slices folder and select Hide All Color Slices . To show all color slices again, right-click and select Show All Color Slices .
  • You can also hide all color slices by disabling the check box for the raster color slice layer in the Layer Manager. This leaves the state of individually hidden raster color slices unchanged (for example, the check box for a hidden class is not enabled again) so when you show the raster color slices layer again, the previously hidden color slice remains hidden.

Edit Color Slices

You can change the settings for the color slices in the Edit Raster Color Slices dialog, which appears when you first select the Raster Color Slices tool. If you have a raster color slices layer already loaded in the Layer Manager and want to edit the settings, open the dialog by right-clicking the Slices folder in the Layer Manager and selecting Edit Color Slices . The Edit Raster Color Slices dialog includes a table and a histogram.

To edit default settings:

  • To edit the default settings, including the number of color slices, the color table, and the minimum and maximum data values to use, click the New Default Color Slices button. The Default Raster Color Slices dialog appears. Change the settings as needed, then click OK .
  • Num Slices sets the number of color slices to apply.
  • Colors provides a drop-down list of predefined IDL color tables.
  • By Min/Max allows you to define the color slices with minimum and maximum data ranges.
  • By Min/Slice Size allows you to define the minimum data range and slice size.
  • Click and drag the right or left edges of a color on the histogram to resize the minimum or maximum data range.
  • Click and drag within a color slice on the histogram to redefine its range.
  • Zoom in to the histogram by dragging and releasing the left and white point sliders below the histogram, or enter Histogram Min and Max values in the fields provided. Click Reset to return to the original histogram view.

To change colors:

  • To apply a different predefined IDL color table to all data ranges, click the Change Color Table button and select a color table from the drop-down list.
  • To change the color for an individual color slice, right-click the color in the table and select Histogram Min . The Edit Slice Color dialog appears for you to define your own color table.
  • Select RGB (red-green-blue), HLS (hue-lightness-saturation), HSV (hue-saturation-value), or CMY (cyan, magenta, yellow) color spaces from the drop-down list.
  • Select Colors from the drop-down list.
  • Enable the check box next to the slider bar of the color space element you want to change. The two remaining sliders represent horizontal and vertical axes of the currently displayed color space and remain constant; only the slider with the enabled check box can change the current color space. Move the slider bars for an element to adjust the values of the color space.
  • To select a color, click in the color palette, or drag the circle selector in the color palette to the desired color.

To refine color slices:

  • To select a color slice, click the table row to highlight it in the histogram, or click the color slice in the histogram to select it in the table row.
  • To delete a color slice, select it in the histogram or from the table row and click the Remove Color Slices button. You can use Shift -click to select multiple sequential rows to delete.
  • To clear all color slices, click the Clear Color Slices button.
  • To manually define individual color slices, click the Add Color Slice button to add a data range to the table and the histogram. If you did not clear all color slices then the new slice is added to the end of the data ranges in the table and the histogram.

To save and restore raster color slices:

You can save the colors and data ranges you defined for raster color slices. You can restore saved raster color slices on the same raster, or on another raster of the same size.

  • To save the raster color slices to a file, click the Save Color Slices to File button and enter a filename.
  • To open a previously saved raster color slices file, click the Restore Color Slices from File button and select the file.

Export Raster Color Slices

You can export the raster color slices to a classification image or to a shapefile.

  • To save a displayed color slices to a classification image, right-click the Slices folder for the raster color slices layer to export and select Export Color Slices > Class Image . The saved file is added to the Data Manager for you to load.
  • To save the displayed color slices to a shapefile, right-click the Slices folder for the layer to export and select Export Color Slices > Shapefile . The saved file is added to the Data Manager for you to load.

Calculate Raster Color Slices Statistics

You can calculate class statistics for one color slice or all color slices. You can plot and report basic statistics including the number of points in a class, and minimum, maximum, mean, and standard deviation for each band for that class. You can also plot and report histograms for each range and display a summary report for all ranges. You can plot and report the following statistics:

  • Basic statistics (the minimum, maximum, mean, and standard deviation for all bands) for your data. Basic statistics are always calculated.
  • Histograms of each band. The frequency distribution calculations also return the number of points, cumulative points, percent for each bin, and cumulative percentage for each DN (digital number) in the image histogram.

To calculate statistics:

  1. In the Layer Manager, right-click a color slice and select Color Slice Stats , or right-click the Slices folder and select Stats for All Color Slices . The Select Class Stats Input File dialog appears.
  2. Select the file to calculate statistics on. If the file is not already open, click Open File and browse to open it.
  3. Click OK . ENVI calculates the statistics and displays them in the Class Statistics Results dialog.

Using the Plot window:

  • By default, the plot displays the minimum, maximum, and mean values. One or all ranges appear, depending on if you calculated statistics for a single color slice or for all color slices.
  • To change the display to standard deviation or histogram, select from the Select Plot drop-down list. If you calculated statistics for all color slice ranges, the drop-down list provides options to display individual color slices or all color slices.
  • If you calculated statistics for all color slices and are viewing color slices individually in the plot, select a different color slices to plot from the Stats for drop-down list.
  • To clear the plot window, click Clear Plot .
  • Use the mouse buttons as follows in the plot:
  • Left Mouse Button : Displays a crosshair cursor on the nearest data point and reports the (x,y) location in the bottom left corner of the plot window
  • Middle Mouse Button : Click and drag to zoom in to the plot. Click to zoom out.
  • Right Mouse Button : Provides menu options listed in ENVI Classic Plot Functions .

Viewing the report text:

  • The report text contains the values of the calculated statistics. It includes file information, the class distribution summary, and the basic statistics and histograms for the selected color slice. If you calculated statistics for all color slices, you can view the basic and histogram statistics for another color slice by selecting from the Stats for drop-down list.
  • The File Info text displays at the top of the report text by default. To go to a different section of the report, use the scroll bar or use the Select Stats drop-down list to jump to a new report section.
  • To copy report text to paste it into another application, select the text to copy, then right-click and select Copy .

The following File menu selections are available from the Class Statistics Results dialog menu bar:

  • Save current Class result to ENVI stats file : Saves the statistics report for the selected color slice to an ENVI statistics file.
  • Save current Class result to text file : Saves the statistics report for the selected color slice to a tab-delimited text file.
  • Save all Class results to ENVI stats file : Saves the statistics report for the all color slices to an ENVI statistics file.
  • Save all Class results to text file : Saves the statistics report for the all color slices to a tab-delimited text file.
  • View Statistics File(s) : Enables you to open one or more ENVI statistics files ( .sta ) and display them in other Statistics Results dialogs. You can use this option to compare your current statistics with information in an another ENVI statistics file.

The following Options menu selections are available from the Class Statistics Results dialog menu bar:

  • Copy results to new window : Creates a copy of the statistics in a new Class Statistics Results dialog so you can compare statistics for different bands.
  • Set Report Precision : Opens the Set Report Precision dialog where you can change the precision of the information displayed in the ROI Statistics Results dialog. This option does not affect histogram precision.
  • Class Area Summary Units : Changes how the units are displayed in the summary report text. This option is available only if the input file has an associated pixel size (from georeferencing or explicit setting of the size in the header file). The default area unit is Meters 2 .
