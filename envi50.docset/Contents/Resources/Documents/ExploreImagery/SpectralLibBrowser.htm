Spectral Library Viewer

Use the Spectral Library Viewer to visualize data from ENVI standard spectral library (SLI) files, THOR Metadata Rich Spectral Library (MRSL) output, and Analytical Spectral Devices (ASD) spectrometers output.

Open a Spectral Library

  1. Select Display > Spectral Library Viewer from the menu bar. The Spectral Library Viewer dialog appears. The viewer also opens if you select a spectral library file using the File > Open option from the menu bar or open a spectral library file from the Data Manager. By default, ENVI opens all spectral libraries in the \spec_lib directory of your ENVI installation.
  2. Click the Open Spectral Library button, then select an MRSL, ASD, or SLI file.
  3. To plot a signature in the viewer, click on it in the Spectral Library Viewer tree. Click the signature again to remove it from the plot. Click additional signatures in the tree to overplot them in the viewer.

Plot Controls

  • Click on the arrow on the right side of the Spectral Library Viewer to expand the dialog panel. The panel shows a list of spectra included in the plot.
  • To change the display properties of the plot or a spectrum, double-click in the plot, or on a specific spectrum in the plot, or on the spectrum listed in the expanded panel. The Properties dialog appears for you to edit the settings.
  • Zoom in and out of the plot by clicking in the plot, then rolling the mouse wheel. Right-click in the plot and select Reset Plot Range to return to the plot to full view.
  • Pan the plot by clicking and dragging in the plot.
  • To display an X-Y line cursor, click in the plot. Then click on spectrum in the plot, or expand the dialog panel and select the spectrum from the list. As you move the cursor over the plot, the line cursor snaps to the closest data point . The x and y values of the point appear in the lower-right corner of the dialog. To close the X-Y line cursor, click outside the plot.
  • To remove all spectra from the plot, right-click and select Remove All . To remove a spectrum or selected spectra from the plot, right-click on a spectrum or use Ctrl +click to more than one, then right-click and select Remove Selected Spectrum .
  • To save the plotted spectra to a spectral library, click the Save to Spectral Library button provided on the expanded dialog panel.

Drag and Drop Spectra to the Spectral Profile

You can drag and drop spectra between the Spectral Profile and the Spectral Library Viewer for analysis:

  1. Extract spectra from an image using the Spectral Profile dialog.
  2. Click the arrow on the right side of the Spectral Library Viewer dialog and the Spectral Profile dialog to expand the dialog panels.
  3. From the Spectral Library Viewer expanded panel, select a spectrum, or use Ctrl +click to multi select.
  4. Drag the selected items to the expanded panel of the Spectral Profile dialog. The spectra appear in the corresponding plot.

Additional Controls

The button controls at the bottom of the dialog do the following:

  • Click Print to print the plot.
  • Click Save As to open the Save As dialog and save the plot.
  • Click Properties to open the Properties dialog and edit the attributes of the plot lines. Double-clicking in the plot also opens the Properties dialog.
  • Click Reset Axis Range to change the range for the x/y axis.
  • Click Edit to select from a drop-down list of options.
  • Click Annotations to select from a list of options to annotate the plot.
