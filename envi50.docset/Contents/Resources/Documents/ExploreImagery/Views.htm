Image Window Views

By default, the Image window displays layers in a single view. You can customize that view by enabling an Overview thumbnail that contains the full extent of the loaded layers, and by dividing the view area into as many as 16 views containing different layers. You can also geographically link multiple Image window views and geographically link the Image window view to ArcMap™.

See the following sections:

  • The Overview
  • Multiple Views
  • Geo Link Multiple Views
  • Geo Link to ArcMap

The Overview

The Overview provides a thumbnail view of all raster layers loaded in the Image window. If you have multiple views in the Image window, each view has its own Overview. When enabled, the Overview appears in the top-left corner of each Image window view. The Overview Locator, a rectangle within the Overview, defines the extent to show in the view. Any rotation or stretch you apply in the Image window view is reflected in the Overview. By default, the Overview shows a snail trail by highlighting areas you have visited in yellow.

  • To show the Overview, enable the Overview check box for the desired View . To hide the Overview, disable the check box.
  • Click and drag inside the Overview Locator to move it to a different area. The view in the Image window shows the corresponding area.
  • Click outside the Overview Locator to go to a different location. The view in the Image window centers at the new location.
  • Click and drag the edge of the Overview Locator rectangle to resize it. The view in the Image window shows the extent defined in the Overview Locator.
  • To hide the snail trail for an Overview, right-click Overview in the Layer Manager and select Show Snail Trail to toggle it off. You can also change the preference setting for Show Snail Trail to hide it by default.
  • To clear the snail trail for an Overview, right-click Overview in the Layer Manager and select Clear Snail Trail .
  • To change the border color of the Overview, such as to make it more visible on the image background, change the preference setting for Default Border Color .

Multiple Views

You can display data in up to 16 different views that are visible at the same time. The zoom and rotate controls work independently for each view, including when the views are geographically linked. Create new views and organize the layout of them from the main menu as follows:

  • Select Views > Create New View to open a new view in the Image window. Repeat to add more views as needed.
  • Select Views > Two Vertical Views to change the layout to two views aligned side by side.
  • Select Views > Two Horizontal Views to change the layout to two views aligned top and bottom.
  • Select Views > 2x2 Views to change the layout to four views in two rows by two columns.
  • Select Views > 3x3 Views to change the layout to nine views in three rows by three columns.
  • Select Views > 4x4 Views to change the layout to 16 views in four rows by four columns.
  • To remove views, in the Layer Manager, right-click on the View to remove and select Remove View . Repeat to remove more views if needed.
  • To revert to a single view, select Views > One View . The active view and any data displayed in it change to a single-view layout and all other views close.

Geo Link Multiple Views

You can geographically link up to 16 views so that all linked views will be centered at the same geographic location. When you go to a certain location in one view, the linked views center at the same geographic location.

To geographically link views, do the following:

  1. Open and display data in different views.
  2. From the main menu, select Views > Geo Link Views . The Geo Link Views dialog displays, showing thumbnails of each view. The views to link must have a base projection and must contain data.
  3. In the Geo Link Views dialog, click on each view to enable the link, or link all applicable views at once by clicking Link All . A Link icon appears on the thumbnail when a view is linked. A color Anchor icon appears on the lead view; all linked views will be centered over a geographic location based on this view. To use a different view for the geographic center location, click on the gray Anchor icon on another thumbnail.
  4. Click OK . All linked views are centered at the geographic center location of the lead view. A Link icon appears in the Layer Manager next to each linked View.

To unlink views in the Geo Link Views dialog, click on each view you want to unlink, or click Unlink All to unlink all views at once, then click OK .

Geo Link to ArcMap

If ArcMap™ is installed, you can geographically link the same georeferenced image, or two images that are georeferenced, between the Image window and the active data frame in the ArcMap data view. Once linked, panning the image in either application will center both views at the same geographic location. You can also move the image by moving the selector in the Overview or by using the Go To tool in ENVI .

Note: The Geo Link to ArcMap option is available when the Image window is in single-view mode only, and is not available in ENVI Print Layout view or in the ArcMap layout view.

Geo Link to ArcMap is available with the following types of georeferencing:

  • Geographic coordinate systems
  • Projected coordinate systems

To use Geo Link to ArcMap:

  1. Open and display the same georeferenced image or two different georeferenced images in ENVI and ArcMap.
  2. From the menu bar, select Display > Geo Link to ArcMap to toggle on.
  • If you have more than one instance of ArcMap running, you will be prompted to select an instance to link to the Image window.

  • If you change the active data frame in ArcMap after establishing a geographic link, you will need to disable and re-enable the Geo Link to ArcMap option in ENVI .

    Note: Geo Link to ArcMap enables dynamic display in ArcMap; the symbology in ArcMap may vary slightly. In addition, ArcMap uses bilinear interpolation while zooming.

To unlink from ArcMap, select Display > Geo Link to ArcMap to toggle off.
