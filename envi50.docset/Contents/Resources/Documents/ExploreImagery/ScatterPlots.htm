Scatter Plots

An ENVI 2D scatter plot displays two bands of image pixel values in a Cartesian plot, with one band providing the x coordinates and the other band providing the y coordinates. If neither band contains dependent data, either band can be plotted on either axis and the scatter plot illustrates only the degree of correlation between the two bands.

By default, the scatter plot only operates on pixels from the displayed portion of the raster.

You can display multiple scatter plots simultaneously.

Create a Scatter Plot

To create a 2D scatter plot:

  1. Open an image file.
  2. Select Display > 2D Scatter Plot on the menu bar. The 2D Scatter Plot dialog appears.

Select Bands

Choose the scatter plot bands from the X Band and Y Band menus. Band 1 is the default choice for the X Band, and Band 2 is the default choice for the Y Band.

Change the Scatter Plot Magnification

You can change the magnification of the scatter plot in several ways:

  • Zoom in and out of the plot by clicking on the plot window, then rolling the mouse wheel.
  • Press Shift +click inside the plot, and move the mouse pointer to draw a bounding box. When you release the mouse button, the area inside the bounding box is magnified to fill the entire plot.

Restrict Scatter Plot to the Viewable Area of the Bands

Select the Viewable Area Only box to limit the scatter plot computations to the visible part of the bands. This option is useful for large files because of the excessive memory required for the calculations. This option is selected by default. For very large files, this option is selected and locked.

The scatter plot uses the displayed pyramid level. For example, if a large data set is displayed and the magnification is reduced to fit the entire data set on the screen, the scatter plot is based on the pyramid level displayed rather than all the pixels of the selected bands.

Display a Density Distribution

A group of image pixels can share the same value in both bands and all will be represented by the same black point on the scatter plot. You can display point density with color by selecting Density Slice . The density color gradient runs from purple (low density) to red (high density).

Display the Pixel Distribution of Scatter Plots

You can turn pixels red in the Image window view that have the same values as points on the scatter plot that lie within a box centered on the mouse pointer. As the pointer moves, the corresponding Image window pixels (called dancing pixels ) change color. You can change the size of the box using the Patch Size slider.

To display dancing pixels in the Image window, move the mouse pointer over the scatter plot and hold down the middle mouse button. You can move the pointer (while holding down the middle mouse button) to interactively display the associated Image window pixels.

Display the Pixel Distribution of Images

You can turn points red on the scatter plot that have the same values as pixels on the Image window that lie within a box centered on the mouse pointer. As the pointer moves, the corresponding scatter plot points (called inverse dancing pixels ) change color. You can change the size of the box using the Patch Size slider.

To display dancing pixels in the scatter plot, move the mouse pointer over the Image window and hold down the left mouse button. The selected pixels are indicated by a red box surrounding the pointer. You can move the pointer (while holding down the left mouse button) to interactively display the associated scatter plot points.

Defining and Exporting Classes

You can select one or more groups of pixels in the scatter plot that highlight the associated pixels in the main image. A group of such pixels is called a class . To create a class of scatter plot pixels:

  • In the Class Functions section of the scatter plot dialog, select a color from the menu. The class' selected pixels and the associated image pixels are set to this color.
  • Define a class region polygon by left-clicking points (or by left-clicking and dragging) on the scatter plot. Right-click to close the polygon, or middle-click to erase the polygon.

Note that the pixels enclosed by the class polygon and the associated pixels in the main image now assume the class color.

After completing a class polygon, you can extend the class by defining more polygons. All the polygons of a particular color are joined to form a single class.

Click Clear to erase the current class.

To define another class, select a different color from the menu and define one or more polygons.

If you select pixels that were included in a previous class, the pixels are assigned to the new class.

Click Export to create a new layer containing the current class pixels. This new layer is added to the Layer Manager, displayed by default, and given the name of the associated class color. If you export multiple classes of the same color, the Layer Manager will assign a unique name for each new layer. If you want to clear or export a previous class, you must first select the previous class color from the menu.

If you click the All Classes box, the Clear and Export buttons apply to all currently-defined classes.

Note: Inverse dancing pixels (pixels highlighted in the main image) override (are shown on top of) class pixels. See Display the Pixel Distribution of Images for more information about inverse dancing pixels.

Additional Dialog Controls

The button controls at the bottom of the dialog do the following:

  • Click Print to print the plot.
  • Click Save As to open the Save As dialog and save the plot.
  • Click Properties to open the Properties dialog and edit the attributes of the plot lines. Double-clicking in the plot also opens the Properties dialog.
  • Click Reset Axis Range to change the range for the x/y axis.
  • Click Edit to select from a drop-down list of options.
  • Click Annotations to select from a list of options to annotate the plot.
