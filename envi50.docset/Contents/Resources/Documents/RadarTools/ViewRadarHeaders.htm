Radar > Open and Prepare Radar Files

View Radar Headers

  1. From the Toolbox, select one of the following:
    • Radar > AIRSAR > View AIRSAR/TOPSAR Header
    • Radar > View COSMO-SkyMed Header
    • Radar > View Generic CEOS Header
    • Radar > RADARSAT > View RADARSAT Header

    2. The Input File dialog appears.

  2. Select the file from which to read the header and click OK .
    • For AIRSAR, select a .stk file. ENVI prints the header information to the AIRSAR File Information dialog. 
  3. To save the report to an output file, select File > Save Text to ASCII from the Header Report window menu bar. The Output Report Filename dialog appears.
  4. Enter an output filename and click OK .
