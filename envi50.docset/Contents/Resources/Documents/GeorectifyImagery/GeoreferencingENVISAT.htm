Georeference ENVISAT

Use Georeference AATSR , Georeference ASAR , or Georeference MERIS to georeference ENVISAT AATSR, ASAR, or MERIS data with the geolocation information included in the ENVISAT file. ENVISAT imagery contains geolocation tie points that correspond to specific pixels in the image. You can use these tie points to automatically georeference the ENVISAT data without building a geometry file.

  1. Open an ENVISAT file by selecting File > Open As > ENVISAT > sensor_type from the menu bar.
  2. Select one of the following items from the Toolbox:
    • Geometric Correction > Georeference by Sensor > Georeference AATSR
    • Geometric Correction > Georeference by Sensor > Georeference ASAR
    • Geometric Correction > Georeference by Sensor > Georeference MERIS
  3. Select an input file and perform optional spatial and spectral subsetting , then click OK . The Select Output Projection dialog appears.
  4. Select the desired output map projection from the list and enter any necessary parameters.
  5. Click OK . The Registration Parameters dialog appears.
  6. Select warping and resampling parameters as needed.
  7. Click OK . ENVI adds the resulting output to the Layer Manager.
