What's New

What's New in This Release

This version of ENVI includes the new and enhanced features listed below. For longtime users of ENVI , ENVI Zoom, and ENVI EX, see The New ENVI for highlights of the new ENVI interface and tools.

High-Performance Display

ENVI enhancements for high-speed raster and vector displays enable you to quickly view and process multiple large datasets, including:

  • Faster initial raster display and pyramid generation.
  • Faster vector display.
  • High-performance navigation, such as pan, fly, zoom, rotate.
  • High-performance enhancements, such as brightness, contrast, sharpen, and stretch.

User Interface

The ENVI user interface includes many enhancements and additions.

New Toolbox

  • ENVI Toolbox: The new ENVI Toolbox integrates tools from ENVI Classic and ENVI .
  • Add Custom ENVI Toolbox Extensions: You can customize your ENVI Toolbox by writing and adding your own tools .

Image Window Enhancements

  • Overview: The Overview provides a thumbnail of the full-extent view of all raster layers loaded in an Image window; if you have multiple Image window views open, each has its own Overview. The Overview includes a Locator rectangle, which enables you to select a specific area of the image to explore.
  • Multiple views in the Image window: You can divide the Image window into as many as 16 different views . Each view can contain multiple layers, and the enhancement tools and the zoom and rotate controls work independently for each view.
  • Geo link multiple views: You can geographically link Image window views so that all linked views will be centered at the same geographic location.
  • Workflow view: When you start a workflow, a separate workflow view is created for the duration of the process. When the workflow process is complete, ENVI returns you to the original Image window view or views. The workflow view is discussed in the help steps for each workflow.

Layer Manager Enhancements

  • Check Boxes to Show/Hide Layers: Enable/disable Check boxes appear next to each layer or band name.
  • View Category: When multiple Image window views are open, a View category in the Layer Manager tree groups the loaded layers according to the view in which they are loaded.
  • Class Layers: If you have a classified image displayed, class layers appear in the Layer Manager tree under their associated raster.

Portal Enhancements

  • View Portal: The View Portal covers the full Image window view and enables you to see through the top layer to the layer loaded below it; you can have one View Portal per view. You can apply blend, flicker, and swipe animation to transition between the top layer and the portal layer.
  • Standard Portal: The Standard Portal is a subset of the Image window view that you can move and resize within the view. Standard Portals support multiple portals within a view. You can apply blend, flicker, and swipe animation to transition between the top layer and the portal layer.

Stretch

  • Stretch Types: Preset Stretch enhancements include different linear percent stretches and the ability to calculate stretch statistics on the entire image or on the view extent only.
  • Custom Stretch: Enhancements to Custom Stretch allow you to adjust individual ranges for RGB in the current layer using a histogram and provide the ability to calculate stretch statistics on the entire image or on the display extent only.

Plots

  • Spectral Library Viewer: The new Spectral Library Viewer enables you to visualize data from ENVI standard Spectral Library (SLI) files, THOR Metadata Rich Spectral Library (MRSL) output, and Analytical Spectral Devices (ASD) spectrometers output. You can drag and drop spectra between the Spectral Library Viewer and the Spectral Profile.
  • Spectral Profile: The new Spectral Profile plots the spectrum of all bands for the selected pixel. You can drag and drop spectra between the Spectral Profile and the Spectral Library Viewer.
  • 2D Scatter Plot: The new 2D Scatter Plot displays two bands of image pixel values in a Cartesian plot, with one band providing the x coordinates and the other band providing the y coordinates.

Reproject Rasters

The new Project Raster tool reprojects georeferenced images to another projected coordinate system using the Esri Projection Engine.

Regions of Interest

  • Regions of Interest Support: You can use ENVI Classic vector ( .evf ) files and ROI ( .roi ) files where Regions of interest (ROI) input is allowed.
  • Convert Vectors to ROIs: You can convert shapefiles ( .shp ) to ROIs and use them as ROI input, and you can save them to an ROI ( .roi ) file.
  • ROI Statistics: Statistics for ROIs are available.

Color Table and Raster Color Slices

  • Color Table: The new Color Table enables you to apply a color table to a grayscale raster band. You can select from pre-defined IDL color tables, and you can work in RGB (red-green-blue), HLS (hue-lightness-saturation), HSV (hue-saturation-value), or CMY (cyan, magenta, yellow) color spaces to define your own color table.
  • Raster Color Slices: The new Raster Color Slices enables you to apply a standard color table to select data ranges.

ENVI Workflows

  • Image Registration: The new Image Registration workflow geometrically aligns two images with different viewing geometry and/or different terrain distortions into the same coordinate system so that the corresponding pixels represent the same objects. An Image Registration Tutorial is also available.
  • Feature Extraction: This workflow has been separated into three distinct workflows: Rule-Based , Example-Based , and Segment-Only . Each workflow includes more powerful segmentation and merge options, a streamlined user interface, and faster processing. For rule-based and example-based workflows, you can select which attributes to use for classification, and you have more options to export results. The example-based workflow now includes a principal components analysis option for supervised classification. For step-by-step examples, see the Rule-Based Tutorial and Example-Based Tutorial . The ENVI_FX_SEGMENTONLY_DOIT , ENVI_FX_RULEBASED_DOIT , and ENVI_FX_EXAMPLEBASED_DOIT routines are also available for programmers to automate each workflow.
  • RPC Orthorectification: The RPC Orthorectification workflow enhancements allow importing ground control points for more accurate orthorectification. It provides options to view the overall model error as a transparent color gradient over the input image and to view error statistics for each ground control point. You can also create a text report that summarizes the details of the orthorectification process. For a step-by-step example, see the RPC Orthorectification Tutorial .
  • Classification: The enhanced Classification workflow allows different settings for thresholding values for each class, and you can optionally output a rule image. For details, see Setting Advanced Parameters and Export Classification Results for details.

ENVI Programming

The new IDL-based API for ENVI allows users to control, extend, and automate the application. For experienced ENVI Classic users, Transitioning to the ENVI API shows how the new ENVI API manages the application, data, and display. The new API includes the following functionality:

  • Control the ENVI application interactively through the IDL command line.
  • Run ENVI in batch mode with no user interaction.
  • Write and deploy custom extensions through the Toolbox.
  • Manage errors.
  • Raster data input, output, and management. The tutorial Working with Raster Layers is available for step-by-step examples of working with ENVI raster routines.
  • Vector data input.
  • Manage and re-order layers and views.
  • A new File > New ENVI Extension wizard in the IDL Workbench. See the IDL Help topic " ENVI Extension Wizard" for details.

Other New and Enhanced Features

  • Spatial Subsetting: Enhancements to spatial subsetting include the ability to interactively select the subset on a thumbnail of the image.
  • Raster Filenames: When saving image files, ENVI now appends .dat to the filename.
  • New ENVI instructional videos are available at the Exelis Visual Information Solutions YouTube channel .
  • Support for BigTIFF Files : ENVI now supports reading TIFF files larger than 4 GB on all platforms, including 32-bit systems.
