The New ENVI

ENVI 5.0 combines the processing and analysis tools of ENVI , ENVI Zoom, and ENVI EX into a single, intuitive interface with a new API. The three-window ENVI of 4.8 is still available and is now called ENVI Classic . ENVI 5.0 represents the major re-architecture and modernization of ENVI . In using it, you will find that your ability to work with multiple, large images will be greatly enhanced. All of the extensions you have written with IDL can be deployed through a toolbox extension in ENVI 5.0 , or continue to work as-is in ENVI Classic .

The default layout of ENVI 5.0 is a single Image window view . You can divide the Image window into as many as 16 separate views; this layout replaces the three-window display groups available in ENVI Classic . Each view has an Overview that you can show or hide. Similar to the Scroll window of ENVI Classic , the Overview provides a thumbnail view of your imagery, and you can select specific areas within the thumbnail to focus on in the larger view. In each Image window view, you can load multiple layers, you can create portals to see down through the top layer to the layer below it, and you can perform independent processing. To explore similar imagery, you can geo-link views so that panning, zooming, and flying actions performed in one view also occur in all geo-linked views.

All ENVI and ENVI Classic tools are accessible through the Toolbox and are grouped by function. A search field is provided at the top of the Toolbox if you wish to find a specific tool. Help for all tools is available when you select Help from the ENVI menu bar.

ENVI 5.0 also includes new programming options that enable you to:

  • Write and deploy custom Toolbox extensions .
  • Run the ENVI application in batch mode with no user interface or user interaction.
  • Issue program calls to control the ENVI user interface.
  • Manage errors in your programs.
  • Extend and customize ENVI with routines .

To start ENVI 5.0 :

  • Windows users: From the Start menu, select All Programs > ENVI 5.0 or ENVI + IDL .
  • From the IDL command line: Type e = envi() .
  • Linux and Macintosh users running X11: Type envi at the shell prompt. To start ENVI Help, type envihelp .

To start ENVI Classic :

  • Windows users: From the Start menu, select All Programs > ENVI 5.0 > Tools > ENVI Classic or ENVI Classic + IDL
  • From the IDL command line: Type envi
  • Linux and Macintosh users running X11: Type envi -classic at the shell prompt.
