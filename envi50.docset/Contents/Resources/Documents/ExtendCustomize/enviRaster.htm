ENVIRaster

This is a reference to a raster object. For details on creating an ENVIRaster, see ENVI::OpenRaster and ENVI::CreateRaster .

Example

The following code opens a file and prints the properties of the created raster dataset.

; Create an ENVIRaster

e = ENVI()

file = FILEPATH('qb_boulder_msi', ROOT_DIR=e.ROOT_DIR, $

SUBDIRECTORY = ['data'])

raster = e.OpenRaster(file)

 

; Print some property values

PRINT, raster

Properties

Properties marked as (Get) can be retrieved, but not set.

AUXILIARY_SPATIALREF (Get)

This property retrieves a reference to the ENVIRasterSpatialRefPseudo , ENVIRasterSpatialRefRPC , or ENVIRasterSpatialRefStandard object, depending on what secondary map information the raster object uses for spatial reference (if any). This property provides a simple way to view secondary map information without having to query the metadata in the file header.

  • If the raster object only has one spatial reference, this property returns a !NULL value.
  • If the raster object contains standard map information plus RPC or pseudo information, the standard map information will be used as the primary spatial reference (returned by the SPATIALREF property) and the RPC or pseudo information will be the secondary spatial reference (returned by the AUXILIARY_SPATIALREF property).
  • If the raster object contains RPC and pseudo map information, the RPC information will be used as the primary spatial reference (returned by the SPATIALREF property) and the pseudo information will be used as the secondary spatial reference (returned by the AUXILIARY_SPATIALREF property).

AUXILIARY_URI (Get)

This property retrieves a string array containing the URIs for the ENVIRaster's associated files, such as the header and pyramid files. If there are no associated files, !NULL is returned.

DATA_TYPE (Get)

Set this property to an integer or string specifying the raster data type (strings are not case-sensitive). When retrieving the property, a string is returned.

String Value Data Type
byte 1 Byte (8 bits)
complex 6 Complex (32 bits)
dcomplex 9 Double-precision complex (64 bits)
double 5 Double-precision floating point (64 bits)
float 4 Floating point (32 bits)
int 2 Integer (16 bits)
long 3 Long integer (32 bits)
uint 12 Unsigned integer (16 bits)
ulong 13 Unsigned long integer (32 bits)

INTERLEAVE (Get)

A string specifying the interleave of the raster.

String Interleave
bil Band interleaved by line
bip Band interleaved by pixel
bsq Band sequential

METADATA (Get)

This property retrieves a reference to the ENVIRasterMetadata object.

NBANDS (Get)

The number of bands per raster.

NCOLUMNS (Get)

The number of columns in the raster.

NROWS (Get)

The number of rows in the raster.

PYRAMID_EXISTS (Get)

Returns 1 (TRUE) if a pyramid file exists for the raster, and 0 (FALSE) if not.

READ_ONLY (Get)

Returns 1 (TRUE) if ENVIRaster is read-only, and 0 (FALSE) if not. If the value is 0, the SetData and Save methods are available.

SPATIALREF (Get)

This property retrieves a reference to the ENVIRasterSpatialRefPseudo , ENVIRasterSpatialRefRPC , or ENVIRasterSpatialRefStandard object.

URI (Get)

A string that is a fully-qualified raster filepath.
