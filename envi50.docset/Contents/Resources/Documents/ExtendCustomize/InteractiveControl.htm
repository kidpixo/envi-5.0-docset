Interactive Control

You can issue program calls at the IDL command line one-by-one to display and process data instead of strictly interacting with the user interface and selecting menu options. This interactive approach is meant for users who want more control over display options or who want to run custom scripts while displaying data in the ENVI user interface.

The following steps show a simple exercise of entering program calls at the command line while also interacting with the user interface.

Start an IDL session and type commands into the IDL or ENVI command line as shown below (in boldface).

  1. Start the ENVI application.

    2. IDL> e = ENVI()

    Tip: Windows users can also start ENVI by selecting Start > Programs > ENVI x.x > ENVI + IDL , which will start the IDL Workbench and the ENVI application.

  2. Define an input image.

    3. ENVI> file = FILEPATH('qb_boulder_msi', ROOT_DIR = e.Root_Dir, SUBDIR = ['data'])

  3. Open the image.

    4. ENVI> image = e.OpenRaster(file)

  4. Obtain the current active view:

    5. ENVI> view1 = e.GetView()

  5. Add an RGB color layer to the current view.

    6. ENVI> rgb = view1.CreateLayer(image)

  6. Add a second view.

    7. ENVI> view2 = e.CreateView()

  7. Add a color-infrared layer to the second view.

    8. ENVI> cir = view2.CreateLayer(image, /CIR)

  8. Geographically link the two views.

    9. ENVI> view1.GeoLink, view2

  9. In the ENVI user interface, select Full Extent from the Zoom To drop-down list.
  10. Click the Pan button and pan around the image.
  11. Remove the first view from the display.

    12. ENVI> view1.Close

  12. Close ENVI .

    13. ENVI> e.Close
