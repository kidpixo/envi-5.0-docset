Processing

Processing routines are used to apply image processing methods to remote sensing datasets. Examples include classification, masking, Band Math, and pan sharpening. Other processing routines perform a specialized function for certain sensor types such as AIRSAR or ENVISAT.

These routines are invoked using the convention ENVI_DOIT, 'xxx_DOIT' , followed by arguments and/or keywords.
