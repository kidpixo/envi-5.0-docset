ENVI_FILE_MNG

This routine has been replaced with the Close method to ENVIRaster and ENVIVector .

This procedure is used to manage ENVI Classic files in memory and on disk.

Syntax

ENVI_FILE_MNG [, / DELETE ], ID = file ID , / REMOVE

Keywords

DELETE

Set this keyword to delete the specified file from the hard disk.

ID

Use this keyword to specify the file ID for the open file. This is the value returned from the keyword R_FID in the ENVI_OPEN_FILE procedure. FID is a long integer with a value greater than 0.

REMOVE

Set this keyword to remove the specified file from within ENVI Classic .

Example

Remove the file specified by FID from ENVI Classic and delete it from the disk.

ENVI_FILE_MNG, id=fid, /REMOVE, /DELETE
