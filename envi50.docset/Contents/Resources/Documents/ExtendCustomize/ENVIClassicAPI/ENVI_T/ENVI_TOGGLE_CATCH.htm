ENVI_TOGGLE_CATCH

This routine has been replaced with the ERROR keyword to most functions and methods. See Manage Errors for more information.

Use this routine to toggle the ENVI Classic error catching handling mechanism on and off. When toggled off the catch mechanism will not redirect IDL programming or execution errors. Instead, errors will be handled according to IDL. This routine is useful when debugging user-developed ENVI Classic routines.

Syntax

ENVI_TOGGLE_CATCH
