ENVI_ADD_PROJECTION

This routine has been replaced with the CreateRasterSpatialRef method to ENVI, followed by the use of the SPATIALREF_OVERRIDE keyword to OpenRaster and the WriteMetadata method to ENVIRaster.

Use this procedure to update the ENVI Classic projections.

Syntax

ENVI_ADD_PROJECTION, Proj [, NAME = string ] [, / WRITE ]

Keywords

NAME

Use this keyword to specify the output file to write, including the directory path. The default is to use map_proj.txt in the map_proj directory of the installation tree. NAME is a string.

WRITE

Set this keyword to update the map projection file. The default is to add the projection to the current ENVI Classic session and not save the projection to a file.
