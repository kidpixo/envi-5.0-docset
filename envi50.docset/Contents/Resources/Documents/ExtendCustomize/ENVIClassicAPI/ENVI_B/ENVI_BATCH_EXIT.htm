ENVI_BATCH_EXIT

This routine has been replaced with the Close method to ENVI.

Use this procedure to terminate an ENVI Classic session from the non-menu batch mode.

Syntax

ENVI_BATCH_EXIT [, / EXIT_IDL ] [, / NO_CONFIRM ]

Keywords

EXIT_IDL (optional)

Use this keyword to force IDL to exit even if ENVI Classic ’s preference setting for Exit IDL on Exit from ENVI Classic is set to No .

NO_CONFIRM (optional)

Set this keyword to override IDL’s exit confirmation preference setting.
