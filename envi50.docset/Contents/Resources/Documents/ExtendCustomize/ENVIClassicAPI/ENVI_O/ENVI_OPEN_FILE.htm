ENVI_OPEN_FILE

This routine has been replaced with the OpenRaster method to ENVI.

Use this procedure to open an ENVI Classic file. The value of R_FID will be the reference for accessing any information about this file.

Syntax

ENVI_OPEN_FILE, Fname [, / INVISIBLE ] [, / NO_INTERACTIVE_QUERY ] [, / NO_REALIZE ] [, R_FID = variable ]

Arguments

Fname

This is the name and path of the file to open.

Keywords

INVISIBLE (optional)

Set this keyword to prevent the file from automatically displaying in the ENVI Classic interface. This keyword only applies when using ENVI_OPEN_FILE in ENVI Classic programming. A file opened with the INVISIBLE keyword set will not appear in the Available Bands List and will not automatically display. See the second Example below.

NO_INTERACTIVE_QUERY (optional)

Set this keyword to suppress bringing up the ENVI Header Information dialog when a valid header file does not exist for Fname. If you set this keyword and a valid header file does not exist, the file cannot be opened and an invalid file ID is returned in the keyword R_FID.

NO_REALIZE (optional)

Set this keyword to suppress opening the Available Bands List. If it is already open, this keyword has no effect.

R_FID (optional)

ENVI Classic library routines that result in new images also have an R_FID, or “returned FID.” This is simply a named variable containing the file ID to access the processed data. Specifying this keyword saves you the step of opening the new file from disk.

Examples

Use ENVI_OPEN_FILE to open a file in ENVI Classic . Check that the file opened properly before using the returned file ID to perform an ENVI_FILE_QUERY .

; Input file definition

fname = '/data/img_001'

envi_open_file, fname, r_fid=fid

if (fid eq -1) then return

envi_file_query, fid, dims=dims, nb=nb

The following code sample illustrates the use of the /INVISIBLE keyword in ENVI Classic programming. The file is opened but not displayed in the ENVI Classic interface.

  1. Start IDL.
  2. Copy and paste the following in the command line:

e = ENVI()

file = FILEPATH('qb_boulder_msi', $

  ROOT_DIR = e.Root_Dir, SUBDIR = ['data'])

ENVI_OPEN_FILE, file, R_FID=fid, $

  /NO_REALIZE, /NO_INTERACTIVE_QUERY, $

  /INVISIBLE

IF fid EQ -1 then RETURN

ENVI_FILE_QUERY, fid, DIMS=dims

PRINT, dims
