ENVI_GET_PATH

This routine has been replaced with the ROOT_DIR property to ENVI.

This function returns the path where the current version of ENVI Classic is installed. There are no arguments or keywords associated with this function.

Syntax

ENVI_GET_PATH()

Examples

At the ENVI command line, use ENVI_GET_PATH with the PRINT command:

print, ENVI_GET_PATH()

Result:

C:\Program Files\Exelis\IDL xx \..\envi xx \classic
