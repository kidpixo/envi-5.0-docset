ENVI_GET_CONFIGURATION_VALUES

This routine has been replaced by the GetPreference method to ENVI.

Use this function to return an anonymous structure of the current ENVI Classic configuration settings for ENVI Classic only. The tag names of the structure are the configuration item, and the values represent the current setting.

Syntax

Result = ENVI_GET_CONFIGURATION_VALUES

Example

This example retrieves the current ENVI Classic configuration values.

cfg = envi_get_configuration_values()
help, cfg, /structure
