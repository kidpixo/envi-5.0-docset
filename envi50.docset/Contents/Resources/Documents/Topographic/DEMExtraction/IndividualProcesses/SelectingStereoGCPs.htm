Select Stereo GCPs

  1. From the Toolbox, select Terrain > DEM Extraction > Select Stereo GCPs . The Select Left Stereo Pair Image dialog appears.
  2. Select GCPs. See View, Add, Edit GCPs for a detailed description of the Stereo GCP Selection for DEM Extraction dialog, and how you can use it to select GCPs.
  3. View the GCP list. Click Show Table in the Stereo GCP Selection for DEM Extraction dialog to open the Stereo GCP List table .
