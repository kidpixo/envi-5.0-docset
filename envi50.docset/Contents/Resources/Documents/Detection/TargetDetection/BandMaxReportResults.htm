Report the Results

After you click Show Report in the Investigate SAM Results panel, the SAM Target Finder with BandMax Summary Report text window appears. This text window reports the following results for the Wizard:

  • Input file (and any spatial subset) for the Wizard
  • Specified targets
  • Specified backgrounds
  • Specified band significance threshold
  • Resulting significant bands subset
  • Output files of the SAM analysis
  • Specified maximum angle parameter for the SAM analysis

Note: You can select File > Save Text to ASCII in the text window to save this information to a text file. When you choose this option, the Output Report Filename dialog appears. ENVI saves the specified file when you click OK .
