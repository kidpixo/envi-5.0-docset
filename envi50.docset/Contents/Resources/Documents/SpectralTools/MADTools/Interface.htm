The MAD Interface

This section describes the main components of the MAD interface. See one of the following sections for more information.

  • Toolbar
  • Overview Window
  • Grid Windows
  • Data Manager
  • Status Bar

Toolbar

Use the various toolbar buttons to explore your imagery or to adjust display settings.

Button

Button Name

Description

Select

Click this button to exit the Zoom, Pan, and Rotate tools.

Pan

Click and drag the cursor in any direction to pan across the image.

Rotate

Click and drag the cursor to rotate in a clockwise or counter-clockwise direction. Or, select an option from the Rotate drop-down list. The choices are 0 , 90 , 180 , and -90 .

Zoom

Click and drag to draw a box around an area to zoom in.

Reset Zoom

Reset the zoom to 1:1 (100%).

Crosshairs

Click this button to overlay crosshairs on all of the grid windows, then click inside a grid window to move the crosshairs over that pixel location. Or, click and drag the crosshairs to a pixel location and release the mouse button to center the crosshairs on that location. To hide the crosshairs, click the Crosshairs button again.

Z Profile

Open a plot window that displays a spectral profile for the pixel underneath the crosshairs. Click another location in a grid window, and the spectral profile will update. If you click and drag the crosshairs to another pixel location, the spectral profile will dynamically update as you move the crosshairs around the image.

Fixed Zoom In

Zoom to the center of the image in each grid window by a fixed percentage. The default is 0.8x.

Fixed Zoom Out

Zoom out from the center of the image in each grid window by a fixed percentage. The default is 0.8x.

Stretch Tools

From the Stretch drop-down list, select a stretch type to apply to all the displayed images at once:

  • Linear: Uses the data minimum and maximum to perform a linear contrast stretch.
  • Linear 0-255: Stretches the range of pixel values from 0 to 255.
  • Linear 2%: Applies a 2% linear stretch to the image. This type of stretch is applied by default to any image that you open.
  • Gaussian: Applies a Gaussian stretch to the image.
  • Equalization: Applies a histogram equalization stretch to the image.
  • Square root: Takes the square root of the image histogram, then applies a linear stretch to the image.

Click the button to apply the selected stretch to the entire image (shown in the Overview window). Or, click the button to apply the selected stretch to the portion of the image displayed in the grid window.

To change the stretch type for an individual grid window, select the grid number under the Grid Windows folder of the Data Manager. Right-click on the Stretch item and select a different stretch type.

Filters

By default, filters are not initially applied to images that you open in MAD. To blur or sharpen all images in the grid windows at once, select an option from the Filter drop-down list:

  • Sharpen [#]: Applies a high-pass convolution filter to the image. Three kernel center options are provided, each with a different percentage of the original data added back to the filtered image: [18] is 90%, [14] is 75%, and [10] is 50%.
  • Smooth [#]: Applies a low-pass convolution filter to the image. Two kernel sizes are provided: 3x3 (slight smoothing) and 5x5 (more smoothing).
  • Median [#]: Replaces the center pixel value with the median value in the kernel, reducing speckling noise. Two kernel sizes are provided: 3x3 and 5x5 .

To change the filter type for an individual grid window, select the grid number under the Grid Windows folder of the Data Manager. Right-click on the Filter item and select a different filter type.

Overview Window

The Overview window shows the full extent of your base layer (the first file that you open) and is useful for navigating large images, The red box inside the Overview window corresponds to the area that is displayed within the grid windows. You can resize the red box or drag it around the Overview window.

To detach the Overview window from the MAD interface, right-click inside it and select Detach Overview . It becomes a floating dialog that you can resize or move to a second monitor. To re-attach the Overview window to the MAD interface, right-click inside the floating dialog and select Attach Overview .

Grid Windows

Separate grid windows display different band combinations of an image or the results of an image processing function. See Opening and Displaying Images for more information on working with grid windows.

Data Manager 

The Data Manager lists the files that you open and makes them available for displaying in the grid windows. The first image that you open is called the base layer and is added to the Loaded Images folder. Any new files that you add to the Data Manager must be derived products and have the same spatial extent of the base layer.

Image layers are listed in the Data Manager with icons that indicate the image type: for multispectral, for panchromatic, and for classification. Expand the image filename to see band, map, or class information. Band names appear with wavelength information, if it is available. Map information is displayed under the Map Info icon . For classification images, class names are listed instead of band names.

See Opening and Displaying Images for more information on opening files.

Status Bar

The Status Bar displays pixel information as you move the cursor over the grid windows or Overview window. For images with no map information, the status bar is initially blank. For georeferenced files, the status bar displays latitude and longitude coordinates.

The status bar will also display the progress of updating imagery or generating RSETs, which are reduced resolution versions of the image used to speed up display and processing.
