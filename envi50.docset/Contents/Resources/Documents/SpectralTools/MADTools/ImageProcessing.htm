Image Processing

MAD provides four image processing tools that you can apply to your base layer. Select the desired tool from the Processing menu of the MAD interface. MAD will immediately process the image and add the resulting image to a new grid window. It also adds the resulting image to the Layer Manager if you want to further analyze the image.

NDVI

Use the normalized difference vegetation index (NDVI) tool to transform multispectral data into a greyscale image that shows healthy, green vegetation as bright pixels. See Calculating Vegetation Indices for more information.

Vegetation Suppression

Use vegetation suppression to remove the vegetation spectral signature from multispectral and hyperspectral imagery, using information from red and near-infrared bands. This method helps you better interpret geologic and urban features and works best with open-canopy vegetation in images with a medium spatial resolution (approximately 30 meters). See Vegetation Suppression for more information.

Terrain Categorization

Use terrain categorization to perform supervised classification that identifies spectral endmembers and their abundances. Similar pixels are then clustered into terrain classes. MAD uses the Sequential Maximum Angle Convex Cone (SMACC) method to calculate spectral endmembers. See Extracting Endmembers with SMACC  for more information.

Other Tools

Select Processing > Band Ratios from the MAD menu bar to use ENVI 's Band Ratio tool. See Band Ratios for more information.

Select Processing > Band Math from the MAD menu bar to run ENVI 's Band Math tool. See Band Math for more information.
