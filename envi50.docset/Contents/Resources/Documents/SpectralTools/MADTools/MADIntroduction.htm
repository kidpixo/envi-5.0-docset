Multi-band Analytic Display

The Multi-band Analytic Display (MAD) tool allows you to view and interact with up to 16 images at once. The images can be different band combinations or results of an image processing function. The MAD interface contains a grid for the images, and the images are dynamically linked so that you can perform identical operations or view pixel information across all the images.
