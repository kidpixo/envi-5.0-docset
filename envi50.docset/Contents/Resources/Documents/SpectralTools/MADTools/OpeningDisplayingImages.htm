Opening and Displaying Images

  1. From the Toolbox, select Multiband Analytic Display , and choose a grid layout option. The choices are MAD 4x4 Grid , MAD 3x3 Grid , MAD 2x2 Grid , MAD 4x1 Grid , and MAD User Configuration , which lets you restore a custom grid layout that you previously created.
  2. Select an image file, and click Open . MAD is designed to work with multispectral imagery, but it accepts any image format that ENVI supports.

    3. The first file that you open becomes the base layer , which defines the map projection and resolution for subsequent images loaded into the display. These images are reprojected to the base layer projection. If subsequent images are not georeferenced, MAD assumes they match the projection and pixel size of the base layer, so it will not resample or reproject the data.

    Because the base layer defines the projection and resolution, you should ideally load images with map information or with higher resolution first. You should not load a classification or rule image first.

  3. To select a different image for the base layer, select File > Load Base Image from the MAD menu bar.
  4. By default, the same image is displayed in all grid windows, but with different band combinations. To change the band combination displayed in a grid window, click inside the window to select it (a red outline will surround it). Right-click inside the window and select Change Band Combination . Select band names for the red, green, and blue channels, and click Load RGB .

Tips for Working with Grid Windows

  • The Grid Windows folder in the Data Manager lists all of the grids that are currently displayed. The band combination, stretch type, and filter type for each displayed image are listed. You can change any of these options for an individual grid window by selecting the grid name and right-clicking on Band , Stretch , or Filter . Then select the desired option.
  • To load a different image in a grid window, click inside the window to select it, and choose File > Add An Image Layer from the MAD menu bar. The new image must be a derived product and have the same spatial extent as the base layer. The new image is listed under the Loaded Images folder in the Data Manager .
  • To add a new grid window to your layout, select Display > Add a New Grid Window from the MAD menu bar. A copy of the base layer (with the same band combination) is added to the new grid window. Change the band combination as needed.
  • To remove a grid window from your layout, click inside the window to select it. Then right-click and select Remove This Display .
  • All grid windows are dynamically linked, meaning that toolbar actions (Pan, Rotate, Zoom, etc.) apply to all the grid windows at once. To unlink a grid window, click inside the window to select it. Right-click and select Unlink From Overview . This "pins" the image in that grid window to its current location, while the other grid windows continue to update as you navigate the image. To re-link the image, right-click and select Link to Overview .
  • To detach a grid window from the MAD interface, click inside the window to select it. Right-click and select Detach from Grid . The image is displayed in a floating dialog that you can resize or move to a second monitor. To re-attach the image to the MAD interface, right-click inside the floating dialog and select Attach to Grid .
  • To enlarge a single grid window to fill up the entire display area of the MAD interface, double-click inside the grid window. Click the Next button to view the next grid window. Right-click menu options and Toolbar functions are still available for use in this display mode. Double-click inside the display to return to the grid layout.
  • As you change a grid's band combination, stretch, or filter type, MAD creates a configuration file ( .xml ) and saves it to your operating system's default temporary directory. You can choose a different location to save the configuration file by selecting File > Save Current Preferences from the MAD menu bar. To later restore your MAD display settings, select Spectral > MAD > User Config from the Toolbox.
