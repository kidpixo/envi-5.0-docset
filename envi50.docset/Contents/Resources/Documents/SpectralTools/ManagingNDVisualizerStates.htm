Manage n-D Visualizer States

Select File from the n-D Controls menu bar to save and restore the state of the n-D Visualizer, including the highlighted groups of pixels.

  • To export the n-D Visualizer state, select File > Save Plot As > PostScript or File > Save Plot As > Image .
  • To print the n-D Visualizer window, select File > Print .
  • To save the n-D Visualizer state, select File > Save State from the n-D Controls menu bar and enter an output filename with the extension .ndv for consistency.
  • To restore a previously saved state, select File > Restore State and select the appropriate file.
