Spectral Tools > Spectral Unmixing > Spectral Hourglass Wizard > Summary Report

Summary Report

  1. In the Investigate SAM Results, Investigate MTMF Results, or Investigate Unmix Results panels, click Next . A Spectral Hourglass Wizard Summary Report appear.
  2. This is the final panel of the Spectral Hourglass Wizard. The summary lists the processing steps you performed and a list of the output files created. Save the summary by selecting File > Save Text to ASCII from the Spectral Hourglass Wizard Summary Report menu bar.
  3. Click Finish to exit the Wizard.
