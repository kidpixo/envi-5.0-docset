Edit Metadata

Use the Edit Metadata panel to add or edit metadata for spectral signatures being imported into a spectral library via the THOR Spectral Library Builder .

The available signatures (derived from the Import Signatures panel) are displayed at the top. Any available metadata for the selected signature are displayed at the bottom.

Adding Fields

  1. To add a new metadata field, select the signature you want to add metadata to.
  2. Click Add Field . You can add a new field for just the selected signature, or for all signatures. The Add Field to Metadata dialog appears.
  3. Choose one of the following options:
    • User Defined: Enter a Field Name (required) and Field Contents (optional).
    • Spectral Feature: This option allows you to define a spectral region with diagnostic features (for example, absorption features) for a given signature. Enter a wavelength range and units in the fields provided. You can perform this step for more than one signature if needed. When you get to the Data Dimensionality and Band Reduction panel later in your workflow, select Diagnostic spectral features in metadata from the Method drop-down list.
  4. Click Accept . The new field is added to the selected signature(s).

Removing Fields

  1. To remove a metadata field, select the signature whose metadata you want to edit.
  2. Select the metadata field to delete.
  3. Click Delete Field . You can delete the field for just the selected signature, or for all signatures.
  4. Click Accept .

Editing Field Names and Content

  1. Select the signature whose metadata you want to edit.
  2. Select the metadata field to edit.
  3. Click View/Edit Field Contents . The View/Edit Field dialog appears.
  4. Edit the field name and contents as desired, then click Accept .
  5. To view the spectrum for any signature, select the signature name and click Plot Spectrum .
