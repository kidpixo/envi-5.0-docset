Edit an Existing Decision Tree

Use Edit Existing Decision Tree to restore an existing decision tree file and edit it as desired.

  1. From the Toolbox, select Classification > Decision Tree > Edit Decision Tree . The Enter Saved Decision Tree Filename dialog appears.
  2. Select the decision tree file to restore and click Open . The selected decision tree appears in the Decision Tree dialog and you can edit it or execute it as desired.
