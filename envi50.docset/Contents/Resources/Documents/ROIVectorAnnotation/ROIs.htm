Regions of Interest

Regions of interest (ROIs) are selected areas of a raster you can use for processing with some ENVI tools, such as to extract statistics for classification. When ROIs are an input option for a tool, you can input ENVI Classic ROI files ( .roi ) , ENVI Classic vector files ( .evf ), and shapefiles ( .shp ).

You can create new ROIs in ENVI Classic . See the topic "Defining ROIs" in the ENVI Classic Help for details.

See the following sections for details:

  • Open, Load, and Manage ROIs
  • ROIs , EVFs, and Shapefiles as Input
  • Convert Vectors to ROIs
  • Save ROIs Converted from Vectors
  • View ROI Statistics

Open, Load, and Manage ROIs

  • Open an .roi , .evf , or .shp file as described in Open and Display Files .
  • ROIs created in ENVI Classic have the same spatial size as their parent raster and cannot be displayed on rasters with a different resolution.
  • To edit the name or color properties of an ROI, right-click the ROI in the Layer Manager and select Properties . Property changes take effect on the edited ROI in the Data Manager, and all versions of it in the Layer Manager.
  • To view statistics for an ROI, right-click the ROI in the Layer Manager and select Stats .
  • To remove all ROIs from the Layer Manager, right-click the Regions of Interest folder and select Remove All . To remove an individual ROI, right-click it in the Layer Manager and select Remove . When you remove ROIs from the Layer Manager, they still remain open in the Data Manager where you can access them for redisplay.
  • You can change the transparency of ROIs with the Transparency tool. In the Layer Manager, select an individual ROI or select the Regions of Interest folder, then apply transparency.
  • If ROIs overlay in the Image window view, you can reorder them in the Layer Manager by using drag and drop.

ROIs , EVFs, and Shapefiles as Input

  • When a tool accepts .roi or .evf files as input, select the file as described in the help steps for that tool. The .roi or .evf file can be open before you start the tool, or you can open the file when you are prompted to input ROIs.
  • When using a .shp file as ROI input, you need to open the shapefile before starting the tool. When you are prompted to input ROIs, the shapefile will be available in the selection list.
  • You can convert the vectors in a shapefile to ROIs.

Convert Vectors to ROIs

You can convert vector records in a shapefile to ROIs. In certain cases, you can display the resulting ROIs on rasters that have different resolutions; to do this, the vectors need to overlay the raster, and the map information must be compatible.

  1. Select the vectors to convert in on of the following ways:

    • Drag and drop a displayed vector layer from the Layer Manager onto the Toolbox option Vector > Convert Vector to ROI .
    • Drag and drop an open shapefile file from the Data Manager onto the Toolbox option Vector > Convert Vector to ROI .

    • Double-click Vector > Convert Vector to ROI in the Toolbox and select the shapefile from the Select Input File dialog.

    The Convert Vector to ROI dialog appears.

  2. Select how to convert the vector records:

    • All records to a single ROI : Converts all vector records in the shapefile to a single ROI layer.
    • Each record to a separate ROI : Converts each vector record to an individual ROI layer.
    • Unique records of an attribute to separate ROIs : If unique records are saved in the shapefile, this option converts vector records of an attribute type to separate layers. For example, vectors with a CLASS_ID of 0 are grouped in one ROI layer, vectors with a CLASS_ID of 1 are grouped in another ROI layer, and so forth. Select the attribute to use from the drop-down list that appears.

    • Specific attribute query : If unique attributes are saved in the shapefile, this option converts vectors of the selected attribute type to a single ROI layer. For example, vectors with the CLASS_NAME water are converted to an ROI layer. Select the attribute to use from the drop-down list that appears.

    The Number of Output ROIs shows how many ROIs will be output, based on your conversion choice.

  3. Converted vectors display in the Image window view by default. Disable the Display Results check box if you do not want the results to display.
  4. Click OK . The ROIs are added to the Data Manger and appear in the Layer Manager as children of the raster. If you have more than one raster open that the ROIs can be associated with, the Select Base Visualization Layer dialog appears. Select the raster to use, then click OK .

Save ROIs

You can save changes you make to ENVI Classic ROIs and to ROIs that were converted from vectors. When you save an ROI that was converted from a vector, it is saved using the spatial size of the parent raster. Once saved, you will not be able to display it on a raster with a different resolution.

  • Save ROIs to the ENVI Classic .roi file format by right-clicking on the Regions of Interest folder in the Layer Manager and selecting Save As .roi .
  • All ROIs listed under the Region of Interest folder in the Layer Manager are saved to one .roi file. If there are ROI layers you do not want to include, remove them from the list before saving.
  • The saved ROIs will be saved with the spatial size of the parent raster and can only be used with a raster of the same spatial size.

View ROI Statistics

To view pixel statistics for ROIs, right-click the ROI in the Layer Manager and select Stats . The ROI Statistics Results dialog appears with a plot window at the top of the dialog and a report text section at the bottom.

Using the Plot window:

  • The default plot display is Min/Max/Mean values for the selected ROIs. To change the plot type, click the Select Plot button and choose from the drop-down list. To clear the plot window, click Clear Plot .
  • Use the mouse buttons as follows in the plot:
  • Left Mouse Button : Displays a crosshair cursor on the nearest data point and reports the (x,y) location in the bottom left corner of the plot window.
  • Middle Mouse Button : Click and drag to zoom in to the plot. Click to zoom out.
  • Right Mouse Button : Provides menu options listed in ENVI Classic Plot Functions .

Viewing the report text:

  • The report text contains the values of the calculated statistics. It shows the resulting type of statistics (basic, covariance, eigenvalues, and histograms) calculated for each band in the image. The text fields include: File Information, Basic Statistics, and Histograms for each band.
  • The default text display is File Info . To change the text display type, click the Select Stats button and choose a new type from the drop-down list.
  • To copy report text to paste it into another application, select the text to copy, then right-click and select Copy .

The following File menu selections are available from the ROI Statistics Results dialog menu bar:

  • Save current ROI result to ENVI stats file : Saves the statistics report to an ENVI statistics file.
  • Save current ROI result to text file : Saves the statistics report to a tab-delimited text file.
  • View Statistics File : Enables you to open a saved statistics file for comparison. You can open multiple statistics files only if they are related (such as, files saved from a group of ROIs with the same output root name).

The following Options menu selections are available from the ROI Statistics Results dialog menu bar:

  • Copy results to new window : Creates a copy of the statistics in a new ROI Statistics Results dialog so you can compare statistics for different bands.
  • Set Report Precision : Opens the Set Report Precision dialog where you can change the precision of the information displayed in the ROI Statistics Results dialog. This option does not affect histogram precision.
