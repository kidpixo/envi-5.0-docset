Convert Vectors to DXF Files

You can convert regions of interest (ROIs), ENVI annotation files, and ENVI vector files (EVFs) to AutoCAD DXF files.

  1. Select one of the following options:
    • Vector > Convert ROI to DXF
    • Vector > Convert ANN to DXF
    • Vector > Convert EVF to DXF
  2. Select an input ROI, annotation, or EVF file and click Open .
    • To add more input files, click Input Additional File and select the desired files.
    • To remove a file from the list, select the filename and click Delete .
    • For annotations, the default output is Pixel Coordinates . If you want to output to map coordinates, select Map Coordinates by clicking on the Output to toggle button.
  3. Enter an output filename and click OK .

  4. If you selected output to Map Coordinates , the Select File Containing Map Coordinates dialog appears.

  5. Select a file that contains the output map coordinates, and click OK .
