Overlay Density Slices

This topic describes how to overlay density slices on an image in an ENVI Classic display group. See Raster Color Slices for the recommended method for creating density slices in ENVI .

Use Density Slice to select data ranges and colors for highlighting areas in a gray scale image. You can use data ranges in the density slice coloring that are from the displayed image or from another image of the same size.

For complete details on using the Density Slice dialog, refer to the topic "Interactive Density Slicing" in the ENVI Classic Help. Here are some basic steps for overlaying density slices on images:

  1. From the Display group menu bar, select Overlay > Density Slice . The Density Slice Band Choice dialog appears. It lists all of the bands that have the same spatial size as the image in the display group.
  2. Select the name of the band to use for the density slice data ranges. The Density Slice dialog appears with eight default ranges listed under Defined Density Slice Ranges . The image's minimum and maximum data values are used to define the ranges. The values show in the Min and Max fields.
  3. Enter the minimum and maximum values in the appropriate fields to change the density slice range. To reset the data range to the initial values, click Reset .
  4. Select whether to apply the density slice colors to the Image window, Scroll window, or both windows by selecting the desired check boxes next to Windows at the bottom of the dialog.
  5. Click Apply to apply the selected ranges and colors to the image.
  6. To remove a range from the list, select the data range and click Delete Range .
  7. To clear the list of density slice ranges, click Clear Ranges .
