Overlay Grid Lines

This topic describes how to overlay grid lines on an image in an ENVI Classic display group. See The Print Layout View for a recommended method for creating maps with grid lines.

Use Grid Lines to overlay one or more grids on an image. You can use grids that are pixel-based or map-coordinate and/or latitude/longitude based (for georeferenced images). Each image display can have its own set of grids, which display in the Image, Scroll, and Zoom windows.

For complete details on using the Grid Line Parameters dialog, refer to the topic "Adding Grid Lines" in the ENVI Classic Help. Here are some basic steps for adding grid lines:

  1. From the Display group menu bar, select Overlay > Grid Lines . The Grid Line Parameters dialog appears and the image is re-displayed with a virtual border if one is not already applied.
  2. Select the desired check boxes next to the Window field at the bottom of the dialog to plot the grid lines in the Image window and/or the Scroll window.

  3. To select the grids to display, click the On/Off toggle buttons next to the labels Pixel Grid , Map Grid , and Geographic Grid . If the image is not georeferenced, then the Map and Geographic coordinate grids are not available. If your image is georeferenced, both the Map and Geographic grids are turned on initially.

  4. In the Grid Line Parameters dialog, enter values in Grid Spacing fields. For pixel-based images, this is the number of pixels. For georeferenced images, this is meters.

  5. For georeferenced images, click DMS <> DD to change between degrees-minutes-seconds and decimal-degrees for the geographic grids. In the Spacing fields, enter the spacing in degrees-minutes-seconds or decimal-degrees.
  6. Click Apply .
