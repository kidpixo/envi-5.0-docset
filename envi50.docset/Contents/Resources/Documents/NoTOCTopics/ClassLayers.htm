Class Layers

When you display a classification image, a Classes folder containing the individual classes appears in the Layer Manager under the image name. In the Image Window view you can overlay the classes on an image and you can show and hide the classes. In the Layer Manager you can select one class or all classes and calculate class statistics on a related file.

  • To hide a class in the view, disable the check box for that class in the Layer Manager. To show the class again, enable the check box.
  • To hide all classes, right-click the Classes folder and select Hide All Classes . To show all classes again, right-click and select Show All Classes .
  • You can also hide all classes by disabling the check box for the classification image in the Layer Manager. This leaves the state of individually hidden classes unchanged (for example, the check box for a hidden class is not enabled again) so when you show the classification image again, the previously hidden class remains hidden.

Calculate Class Statistics

You can use the classification results of the displayed image to calculate statistics for any related input file. You can plot and report basic statistics including the number of points in a class, and minimum, maximum, mean, and standard deviation for each band for that class. You can also plot and report histograms for each class and display a summary report for all classes. You can plot and report the following statistics:

  • Basic statistics (the minimum, maximum, mean, and standard deviation for all bands) for your data. Basic statistics are always calculated.
  • Histograms of each band. The frequency distribution calculations also return the number of points, cumulative points, percent for each bin, and cumulative percentage for each DN (digital number) in the image histogram.

To calculate statistics:

  1. In the Layer Manager, right-click a class and select Class Stats , or right-click the Classes folder and select Stats for All Classes . The Select Class Stats Input File dialog appears.
  2. Select a file of the same spatial size as the classification file. If the file is not already open, click Open File and browse to open it.
  3. Click OK . ENVI calculates the statistics and displays them in the Class Statistics Results dialog.

Using the Plot window:

  • By default, the plot displays the minimum, maximum, and mean values. One or all classes appear, depending on if you calculated statistics for a single class or for all classes.
  • To change the display to standard deviation or histogram, select from the Select Plot drop-down list. If you calculated statistics for all classes, the drop-down list provides options to display individual classes or all classes.
  • If you calculated statistics for all classes and are viewing classes individually in the plot, select a different class to plot from the Stats for drop-down list.
  • To clear the plot window, click Clear Plot .
  • Use the mouse buttons as follows in the plot:
  • Left Mouse Button : Displays a crosshair cursor on the nearest data point and reports the (x,y) location in the bottom left corner of the plot window
  • Middle Mouse Button : Click and drag to zoom in to the plot. Click to zoom out.
  • Right Mouse Button : Provides menu options listed in ENVI Classic Plot Functions .

Viewing the report text:

  • The report text contains the values of the calculated statistics. It includes file information, the class distribution summary, and the basic statistics and histograms for the selected class. If you calculated statistics for all classes, you can view the basic and histogram statistics for another class by selecting from the Stats for drop-down list.
  • The File Info text displays at the top of the report text by default. To go to a different section of the report, use the scroll bar or use the Select Stats drop-down list to jump to a new report section.
  • To copy report text to paste it into another application, select the text to copy, then right-click and select Copy .

The following File menu selections are available from the Class Statistics Results dialog menu bar:

  • Save current Class result to ENVI stats file : Saves the statistics report for the selected class to an ENVI statistics file.
  • Save current Class result to text file : Saves the statistics report for the selected class to a tab-delimited text file.
  • Save all Class results to ENVI stats file : Saves the statistics report for the all classes to an ENVI statistics file.
  • Save all Class results to text file : Saves the statistics report for the all classes to a tab-delimited text file.
  • View Statistics File(s) : Enables you to open one or more ENVI statistics files ( .sta ) and display them in other Statistics Results dialogs. You can use this option to compare your current statistics with information in an another ENVI statistics file.

The following Options menu selections are available from the Class Statistics Results dialog menu bar:

  • Copy results to new window : Creates a copy of the statistics in a new Class Statistics Results dialog so you can compare statistics for different bands.
  • Set Report Precision : Opens the Set Report Precision dialog where you can change the precision of the information displayed in the ROI Statistics Results dialog. This option does not affect histogram precision.
  • Class Area Summary Units : Changes how the units are displayed in the summary report text. This option is available only if the input file has an associated pixel size (from georeferencing or explicit setting of the size in the header file). The default area unit is Meters 2 .
