Spatial Subsetting

Use spatial subsetting to limit applying a function to a spatial subset of the image. In the Select Input File dialog, click Spatial Subset . The Spatial Subset dialog appears.

  • Enter beginning and ending sample numbers in zero-based file coordinates in the Samples and to fields.
  • Enter beginning and ending line numbers in zero-based file coordinates in the Lines and to fields.
  • Enter a value in the NS and NL fields to adjust the ending numbers for the Samples and Lines fields accordingly.
  • Interactively select the subset on the image thumbnail. If a pyramid file exists for the selected raster, a thumbnail appears in the Spatial Subset dialog (if a pyramid file does not exist, the outline of the raster appears). Click and drag in the thumbnail to draw a box around the area to subset. You can move and resize the box, if needed. You can also define the subset in the Samples and Lines fields.
  • Click Use View Extent to use the portion of the image visible in the Image window view as the subset. Use View Extent is available when the image to subset is in the active view and it is visible in the view. It is not available if the image has been rotated, or if it has arbitrary pixel-based projection.
  • Click Use Full Extent to use the entire image.
  • Click Subset By File to use an image subset from another file. The subset file must have spatial extents that overlap with the base file and cannot have arbitrary, pseudo or pixel-based projection. The Select File Input dialog appears for you to navigate and choose the file.

Spectral Subsetting

Use spectral subsetting to limit applying a function to selected bands of the image. In the Select Input File dialog, all of the bands are selected by default, which specifies no subsetting.

  • To one or multiple band bands to use, right-click on the bands and select Select None , then on the desired band, or use Ctrl or Shift + click to multi-select bands.
  • Use Ctrl +click to select non-adjacent bands.
  • Use Shift +click to select a range of bands.
  • Click Select All to select all bands (no subsetting).
  • Click Select None to clear band selections.
