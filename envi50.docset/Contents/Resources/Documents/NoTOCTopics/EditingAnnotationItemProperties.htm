Annotation Item Properties

You can change the properties of an annotation item after you create it.

To access the properties, right-click in the Layer Manager on the annotation item you want to modify and select Properties . You can also click on the annotation item in the Image window view, then right-click and select Properties .

The Properties dialog appears, listing settings specific to the annotation item type. The example below shows settings specific to a text annotation item.

The following describes the annotation properties you can change, listed alphabetically. Some of the properties are used by more than one annotation item type.

Property

Description

Arrow Head Fill

Enables or disables filling the interior of the arrow head for an arrow annotation.

Arrow Head Size

The size of the arrow head for an arrow annotation, in points.

Arrow Heads

The angle of the arrow head for an arrow annotation, in degrees.

Background Color

The background color for a text or symbol annotation.

Background Fill

The background fill color for a text or symbol annotation.

Character

The character used for a symbol annotation.

Color

The color of a text or symbol annotation.

Fill Color

The fill color of a polygon, rectangle, or ellipse annotation.

Fill Interior

Enables or disables filling the interior of a polygon, rectangle, or ellipse annotation.

Font Name

The font used for a text or symbol annotation.

Font Size

The font size for a text or symbol annotation.

Font Style

The font style (e.g., bold) for a text annotation.

Horizontal Alignment

The horizontal alignment of a text annotation.

Line Color

The line color for a polyline, arrow, polygon, rectangle, or ellipse annotation.

Line Style

The line style (e.g., dashed line styles) for a polyline, arrow, polygon, rectangle, or ellipse annotation.

Line Thickness

The thickness of lines on a polyline, arrow, polygon, rectangle, or ellipse annotation.

Rotate with View

Enables or disables rotating the annotation along with the data in the Image window.

Scale on Zoom

Enables or disables scaling the text, symbol, or picture annotation size according to the zoom level. If set to True , the item's size increases or decreases accordingly when the zoom level increases or decreases. If set to False , the item's size remains the same, regardless of the zoom level.

Show

Enables or disables showing the annotation in the Image window.

Show Outline

Enables or disables showing the outline of a polygon, rectangle, or ellipse annotation.
